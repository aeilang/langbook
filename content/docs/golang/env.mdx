### 1.安装 Go [#install]

去<a href="https://go.dev/dl/" target="_blank">
官网</a>下载安装包, 官网有<a href="https://go.dev/doc/install" target="_blank">
安装指南</a>。

验证安装, 打开命令行，输入

```sh
go version
```

看到版本信息就安装成功了。

### 2. 配置 vscode [#vscode]

你也可以使用 `Goland`。在插件市场搜索 Go, 安装支持 Go 的插件:

![](@/public/images/vscode.png)

新建一个`main.go`文件， vscode 左下角会弹出下载分析工具的标识，如果不弹出你也可以把鼠标悬停到左下角`{} Go`处:

![](@/public/images/goenv1.png)

点击`update` 会弹出

![](@/public/images/goenv2.png)

全选进行下载即可。这些是 go 进行代码检查和自动格式化的一些工具。

### 3. 配置代理 [#proxy]

有时因为网络原因下载不了依赖，我们可以设置代理来解决。打开命令行，运行以下命令:

```shell
go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct
```
